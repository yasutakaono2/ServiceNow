# ServiceNow
ServiceNowとGithubのrepository連携、ソースコード連携のテスト

# 実行方法
1. Github側の設定
1.1 repositoryの作成
1.2 2FAの設定
1.3 AccessToken Keyの作成

2. ServiceNow側の設定
2.1 credentialの作成
2.1.2 Github repositoryとAccessToken Keyの記入(Basic Auth)
2.2 studio の起動
2.3 studio のソースコントロールで連携

3.Test実行
3.1 client script作成
3.2 commit changes
3.3 Create Pull Request
